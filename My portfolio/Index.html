About Me

Hello! I'm Carlson Wachira, a student at USIU currently studying APT. Welcome to my portfolio website where you can find information about my projects and skills. The Website includes assignments issued throughout the semester for the APT 1030 course. They involve the designing of programs that perform various tasks. My hope is that creating this portfolio will aid in the gaining of more skills as i strive towards becoming a Web Designer.

Projects

Login Interface

The first assignment involved the designing of a login interface using different programming languages. The user would be prompted to enter an email and a password in order to initiate logging in. The email and password of the user have to be valid in order for them to be granted access.

SWISH Prolog Family Tree

The second assignment involved the use of facts, relationships and logic to create a family tree of four generations using SWISH Prolog.

Area of Shapes

The assignment required students to create a program that would prompt a user to enter the name of a shape. The user would also be required to input values that would be assigned to the shape selected. Based on these values, the program would then calculate the area of the shape and display it as output.

Grade Program

The assignment required students to design a grading program for USIU that would prompt the user to enter their total marks. The program would then display their final grade based on their input.
